# ⚡ CSS Pikachu

A fun and creative project that recreates **Pikachu** using only **HTML** and **CSS**. No images, SVGs, Canvas, or JavaScript are used—every part of the character is built with CSS shapes, borders, border-radius, positioning, and pseudo-elements.

## ✨ Features

* 🎨 Pure HTML & CSS artwork
* ⚡ No external libraries or frameworks
* 👂 Detailed ears with black tips
* 👀 Realistic eyes with highlights
* 😊 Cute smiling mouth with tongue
* ❤️ Classic Pikachu red cheeks
* 📱 Responsive page layout with centered design

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/CSS-Pikachu.git
```

2. Open the project folder.

3. Double-click `index.html` or open it in your favorite browser.

That's it! 🎉

## 🎯 Learning Objectives

This project demonstrates:

* CSS positioning (`absolute`, `relative`)
* CSS pseudo-elements (`::before`, `::after`)
* Border radius techniques
* Layering with `z-index` and HTML order
* Creating illustrations without images
